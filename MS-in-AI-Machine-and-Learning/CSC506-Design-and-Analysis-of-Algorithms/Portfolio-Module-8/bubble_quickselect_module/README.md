## Module Purpose

This package is the Portfolio Module 8-native feature area. It demonstrates:

- Bubble Sort for integer lists with step-by-step adjacent comparison traces.
- Quickselect for finding the kth-smallest integer without fully sorting.
- `CourseSet`, a hash-backed Set ADT with static and dynamic operations.
- Benchmarks comparing Bubble Sort, Python `sorted`, Quickselect, and full-sort selection.
- Written analysis connecting Module 8 concepts to CSC506 algorithm trade-offs.
- A root-rendered Streamlit page that runs inside the single Portfolio Module 8
  app.

## How to Run

From the repository root:

```bash
source venv/bin/activate
streamlit run Portfolio-Module-8/streamlit_app.py
```

Open the `Bubble Quickselect Sets` tab in the root app. This module
intentionally has no separate `streamlit_app.py` entrypoint.

## How to Test

```bash
source venv/bin/activate
python -m pytest Portfolio-Module-8/tests -v
```

## Integration Notes

The integer dataset helpers and set operations import independently from the
larger integrated Portfolio Module 8 packages. Graph and hash-priority helper
imports are lazy so Bubble Sort, Quickselect, Set tests, and this page can run
before the full portfolio integration packages are populated.

The root app imports
`bubble_quickselect_module.overview_page.render_bubble_quickselect_sets_page()`
and renders it directly inside the `Bubble Quickselect Sets` tab. The page
starts with an `Overview` landing tab, then preserves the Set lab, Bubble Sort
lab, Quickselect lab, benchmark lab, and written analysis workflows.
